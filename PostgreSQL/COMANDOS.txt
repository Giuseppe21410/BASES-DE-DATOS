## Se adjuntan un conjuntos de comandos que nos permitiran interactuar en terminal con PostgreSQL:

- "psql --username=username --dbname=name": Inicia la aplicación.

- "\l": Ver el conjunto de bases de datos. 

- "CREATE DATABASE database_name;": Permite crear bases de datos.

- "\c database_name": Conectar con una base de datos.

- "\d": Muestra las tablas de una base de datos. / "\d table_name": Ve más en detalle el contenido de una tabla.

- "CREATE TABLE table_name();": Crea una tabla.

- "ALTER TABLE table_name ADD COLUMN column_name DATATYPE;": Añade una columna para un tipo de dato específico en una tabla.
   Algunos DATATYPE son: 
     - "INT":Números enteros.
     -  "VARCHAR(max_value)": Cadena de texto de tamaño máximo. Se insertan datos con comillas simples '...'.

- "ALTER TABLE table_name DROP COLUMN column_name;": Para eliminar una columna dentro de una tabla.

- "ALTER TABLE table_name RENAME COLUMN column_name TO new_name;": Renombra una columna.

- "INSERT INTO table_name(column_1, column_2,...) VALUES (value1, value2,...);": Añade filas (Contenido de las columnas) a la tabla. 

- "SELECT columns(*:ALL) FROM table_name;": Ver el contenido en una fila de una tabla.

- "DELETE FROM table_name WHERE condition;": Elimina una fila a partir de una condición.

- "DROP TABLE table_name;": Elimina una tabla.

- "\q": Para salir.

- "ALTER DATABASE database_name RENAME TO new_database_name;": Cambia el nombre de una base de datos.
